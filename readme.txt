movCase  (于2016年7月)
	
	一个包含图表、数据列表等展示方式的移动端页面

	特点：
	1、rem布局方案；
	2、原生js、canvas结合实现带动画效果饼图；
	3、原生js实现页面列表模块的滑动切换，并保证切换展示不受列表高度不一影响；
	4、原生js实现三个列表的分别加载更多效果；	

yikouyuanye   (于2016年7月)

	一个基于jquery的pc端项目，主要用于展示一口原野燕麦浆产品
	
	页面主要效果：

	1、首屏打开方式；首屏满屏展示；
	2、首屏滚动隐藏时页面元素的飞出凸显立体效果；
	3、第二页特色图片鼠标滑过时蒙版跟随鼠标进入和滑出方向显示和隐藏；
	4、滚动到页面顶部。

chart       (于2015年)
	
	一个原生js封装的适用于手机端的股票分时图插件
	
	特点：
	1、利用canvas和原生js实现，外观较为美观；
	2、支持一页多个图表并存；
	3、支持触摸显示详细数据；
	4、不刷新页面可自动画出新增数据（每30秒取一次新数据）；
	5、调用接口简单，可自定义图表外观颜色。

select     （于2012年）

	一个原生js封装的自定义select外观插件，主要解决select外观美观问题，及ie6下select层级太高无法被遮罩遮盖问题。

	特点：
	1、页面布局无需特殊处理，仅根据需要编写样式即可；
	2、调用方便，无需繁琐配置，只需简单传入对应元素name即可；
	3、支持一个页面多个select，互不冲突；
	4、支持键盘旋转；
	5、支持多选；
	6、支持元素自定义的onchange事件。
	7、仍有不完善的地方。
